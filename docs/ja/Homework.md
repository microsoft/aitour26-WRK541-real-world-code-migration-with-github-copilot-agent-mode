# ホームワーク

ワークショップの修了、おめでとうございます！学習を定着させ、さらなる概念を探求するためのホームワークです。

### テイクホームチャレンジ: Azure にアプリケーションをデプロイする

コンテナ化を含む前のステップをすべて完了している場合は、Azure へのデプロイに挑戦してみましょう。Azure アカウントと、デプロイに関する質問やガイダンスを得るための [GitHub Copilot for Azure 拡張機能](https://marketplace.visualstudio.com/items?itemName=ms-azuretools.vscode-azure-github-copilot) が必要です。

![GitHub Copilot for Azure 拡張機能のページ](./media/gh-copilot-for-azure.png)

1. Azure アカウントにサインインした状態で `@azure` を使って、コンテナ化した C# アプリのデプロイ方法を質問する
1. 提案されたサービスの中から一つ選んでコンテナをデプロイする（例: Azure Container Apps または Azure App Service）
1. コンテナがデプロイされて正常に動作することを確認する

作業が終わったら、不要なクラウドコストが発生しないよう、作成したリソースをすべて削除してクリーンアップすることを忘れずに。
