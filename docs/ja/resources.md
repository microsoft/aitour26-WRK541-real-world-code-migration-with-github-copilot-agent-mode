## ローカル環境の前提条件

GitHub Codespaces を使わずにローカル環境でワークショップを進める場合は、以下がインストール済みであることを確認してください（演習にすぐ取り掛かれるよう、事前にセットアップを済ませておくことを前提としています）:

- VS Code で GitHub Copilot が有効化された GitHub アカウント
- リポジトリのクローン用 Git
- Visual Studio Code
- Python 3.12 と `pip`（`src/python-app/requirements.txt` のインストールに必要）
- .NET 10 SDK（C# Minimal API 用）
- 任意: VS Code からエンドポイントをテストするための REST Client 拡張

インストール手順と確認ステップの詳細については、リポジトリ内の前提条件チェックリストを参照してください: [ローカル開発の前提条件](../../PREREQUISITES.md)
