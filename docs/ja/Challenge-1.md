# ボーナスチャレンジ

さらなる挑戦の準備はできましたか？このセクションでは、ここまでの成果をさらに発展させる応用的なシナリオを用意しています。

## チャレンジ 1: プロジェクトのコンテナ化

**難易度:** 🔥 中級

依存関係のインストールやコンパイルなしに他の環境でもプロジェクトを実行できるよう、コンテナ化してみましょう。

Agent モードを使って Dockerfile を作成しましょう。

1. C# プロジェクトをコンテナ化するための Dockerfile 生成を Copilot に依頼する
1. 生成された Dockerfile を確認し、必要に応じて Agent モードを使って修正を加える
1. Agent モードを使ってコンテナがビルドできて期待通りに動作することを確認する

コンテナが正しく動作するよう、必要に応じて調整してください。
